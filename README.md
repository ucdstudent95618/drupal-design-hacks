## Buttons

In order to create a button you must do the following

1. Create a link in the editor:

   ![Button Editor](\img\btn-2.png)

HTML Code:

```html
<a href="http://www.google.com">button</a>
```

2. Add CSS

```css
display: block;
background-color: blue;
width: 175px;
height: 45px;
line-height: 45px;
color: white;
border-radius: 10px;
font-size: 18px;
text-decoration: none;
```
